Conceptos básicos de JavaScript

Aprende sobre selectores, eventos y funciones en una forma sencilla.

Selectores

Los selectores en JavaScript sirven para acceder a los elementos del HTML desde el código. Con ellos podemos cambiar texto, estilos o valores.

  • document.getElementById()
  • document.querySelector()
  • document.querySelectorAll()

Este texto cambiará con un selector.

Eventos

Los eventos permiten que el navegador “reaccione” a las acciones del usuario, como hacer clic, mover el ratón o escribir.

  • click – Al hacer clic en un elemento
  • mouseover – Al pasar el ratón
  • keyup – Al soltar una tecla

Funciones

Las funciones son bloques de código que realizan una tarea específica y pueden reutilizarse cuando se necesite.

function saludar(nombre) {
  alert("Hola " + nombre);
}
